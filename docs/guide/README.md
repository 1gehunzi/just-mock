# 介绍

Just Mock 是一个专门面向 Web 项目的 Mock 工具。你可以使用 Just Mock 来模拟后端响应（用于前后端联调、QA 测试、PM 验收、线上问题还原等场景），仅需安装浏览器插件即可满足 90% 的 Mock 需要。

Just Mock 诞生的初衷是为了解决前端 `造数` 老大难问题。
